## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## API

Install the json-server lib to run the API `npm install -g json-server`
Once the installation completes, open the server folder and run the command `json-server db.json`
The server will be on the port 3000
